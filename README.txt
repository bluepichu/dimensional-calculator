This is a calculator made for dimensional analysis.  Given text input, it:

-Parses the input
-Outputs the input in LaTex format (rendered by MathJax)
-Calculates the answer (including units) and outputs in LaTex format

Later features will include:

-More units
-Contstants (pi, e, R, ...)
-Variables (store custom variables)
-Printable output
